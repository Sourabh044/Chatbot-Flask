# ChatBot - python

## Creating vitual environment

```sh
python3 -m venv .venv
```

## activate the environment

```sh
source .venv/bin/activate
```

## install the requirements

```sh
pip install -r requirements.txt
```

## run the bot
```
python app.py
```


Now you can chat with the bot.
